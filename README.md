# Manipulação de Arquivos com Pathlib (Python)

Coleção de exemplos práticos usando a biblioteca `pathlib` para manipulação de arquivos e diretórios em Python.

## 🧠 O que é o Pathlib?
Biblioteca padrão do Python para trabalhar com caminhos de forma orientada a objetos, substituindo `os.path` com mais:
- Legibilidade
- Segurança
- Portabilidade

## 🚀 Para que serve?
Automatizar tarefas como:
- Organização de diretórios
- Processamento de arquivos
- Gerenciamento de mídia
- Compactação e extração
- Limpeza de arquivos

## 📚 Exemplos
Cada arquivo aborda um tema:

- **Ex01**: Conceitos básicos (`Path`, `exists`, propriedades, `iterdir`)
- **Ex02–Ex05**: Renomeação de arquivos (simples, com contexto, hierarquia e data)
- **Ex06**: Alteração de extensão (`with_suffix`)
- **Ex07**: Criação de arquivos (`touch`)
- **Ex08–Ex09**: Compactação e extração (`zipfile`)
- **Ex10**: Busca de arquivos (`rglob`)
- **Ex11**: Remoção de arquivos (`unlink`)

## ⚠️ Boas práticas
- Verificar se é arquivo (`is_file()`)
- Evitar sobrescritas desnecessárias
- Cuidado com exclusões (`unlink()`)
- Validar arquivos ZIP
- Evitar caracteres inválidos
