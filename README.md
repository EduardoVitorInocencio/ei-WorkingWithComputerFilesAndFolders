# 📂 Manipulação de Arquivos com Pathlib (Python)

Este repositório mostra exemplos práticos usando `pathlib` para trabalhar com arquivos e pastas em Python.

## 🧠 O que é Pathlib?

`pathlib` é uma biblioteca padrão que permite manipular caminhos de forma mais simples e organizada (orientada a objetos), substituindo `os.path`.

### Vantagens:

* Código mais limpo
* Mais seguro
* Funciona bem em diferentes sistemas

## ⚠️ Boas práticas

* Verifique se é arquivo antes de operar (`is_file()`)
* Evite sobrescrever arquivos
* Tenha cuidado ao deletar (`unlink()`)
* Valide arquivos ao extrar ZIP
